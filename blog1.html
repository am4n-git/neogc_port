    Neumorphic and Glassmorphic Design

    Few days back I saw a thread on twitter which helped me to style minimalistic forms with some cool UI styling. It is a stylish UI style nowadays. It's worth noting that neumorphism and glassmorphism is'nt a replacement for classic design style, but an addition to app use interface design.



    Neumorphism


    Steps to create a neumorphic form using few simple CSS steps :


    1. Create a container div and set background color darken than white . e.g #ecf0f3. Ensure that the color of body and container should be same.

    2. Also ensure left and top border of container should be brighter than right and bottom border. This can make the element appear like it has the light source from the left corner of the screen.




    3. The input fields will have a sunken effect. To achieve this we will use inset box-shadow. As light is coming from top-left corner, hence right and bottom border will be bright in this case because input fields have sunken effect.




    4. Create a button prefereably of same color as logo because this will give you a more fascinating look to your design. Addition to it we can add hover selector and apply box-shadow none when we mouse over it.

    Neumorphic design is based on box-shadow property. we just nedd to remember two things down pointing backhand index-
    (a). Normal box shadow
    (b). Inset box shadow.

    You can also create neumorphic design using any combination you just need to remember that shades should'nt differ much.




    Glassmorphism


    Glassmorphism is a unified name for the popular frosted Glass design. It is trendy web design nowadays which looks super cool

    Let's create a glassmorphic form using few simple CSS steps :


    1. Starting of the fundamental structure, Just start with writing all necessary tags for form layout. You can make number of input field according to you.



    2. Create glass
    - Add thin white border
    - Low opacity white background color
    - Low opacity and blur black shadow




    3. Style input field and button This is similar to step 2. We need to add glass look in the input field.




    4. Add some final detailing
    - One consistent text color
    - align "forgot password" and "signup"
